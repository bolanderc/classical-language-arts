# Example Portfolio Flow

This folder is a local mock of the Google workflow.

It shows what the pieces would look like after a few assignments from *The Secret Garden*:

1. Emma and James each have a response packet.
2. The manifest marks selected work for portfolio inclusion.
3. The generated portfolios summarize completed work and link back to packet sources.

## Files

- `manifest-example.csv`: example rows for the tracking spreadsheet
- `student-packets/emma-secret-garden-packet.md`: sample packet content for Emma
- `student-packets/james-secret-garden-packet.md`: sample packet content for James
- `generated-portfolios/emma-portfolio-spring-2026.md`: example portfolio output
- `generated-portfolios/james-portfolio-spring-2026.md`: example portfolio output

## What to Review

Look at the generated portfolio files first. Those are the parent-facing outputs.

Then look at the student packets and decide:

- Is the packet too long or too structured?
- Should portfolios include excerpts, links only, or both?
- Should spelling, writing, and projects be separated?
- Do you want parent notes in the packet, the manifest, or both?
